1. はじめに

Qt-Theme-Studioは、Qtアプリケーション向けの統合テーマエディターです。直感的なビジュアルインターフェースを使用して、美しくアクセシブルなテーマを効率的に作成できます。

主な機能

  • 統合テーマエディター: 色、フォント、プロパティの直感的な編集
  • ゼブラパターンエディター: WCAG準拠のアクセシブルなテーマ作成
  • ライブプレビュー: リアルタイムでのテーマ確認
  • テーマ管理: インポート、エクスポート、テンプレート機能
  • 多形式対応: JSON、QSS、CSS形式での入出力

対応フレームワーク

  • PySide6（推奨）
  • PyQt6
  • PyQt5

2. インストールと起動

システム要件

  • Python 3.8以上
  • PySide6、PyQt6、またはPyQt5のいずれか
  • qt-theme-managerライブラリ

インストール手順

重要: qt-theme-managerライブラリは必ずGitHubリポジトリから直接インストールしてください。 
# 1. Qt-Theme-Studioのインストール
pip install -e .

# 2. qt-theme-managerライブラリのインストール（必須）
pip install git+https://github.com/scottlz0310/Qt-Theme-Manager.git

# 3. Qtフレームワークのインストール（いずれか一つ）
pip install PySide6  # 推奨
# または
pip install PyQt6
# または
pip install PyQt5

アプリケーションの起動

以下のいずれかの方法でアプリケーションを起動できます：

# 統合ランチャーを使用
python launch_theme_studio.py

# モジュールとして実行
python -m qt_theme_studio
ヒント: 初回起動時には、利用可能なQtフレームワークの自動検出が行われます。

3. インターフェース概要

メインウィンドウの構成

Qt-Theme-Studioのメインウィンドウは以下の要素で構成されています：

  • メニューバー: 主要な機能へのアクセス
  • ツールバー: よく使用する機能のクイックアクセス
  • 作業エリア: エディターとプレビューの表示領域
  • ステータスバー: 現在の状態と操作結果の表示

メニュー構成

メニュー 主な機能
ファイル 新規作成、開く、保存、エクスポート、インポート
編集 Undo/Redo、設定
表示 プレビュー、ツールバー、ステータスバーの表示切り替え
ツール ゼブラパターンエディター、テーマギャラリー
ヘルプ ヘルプ、ユーザーマニュアル、バージョン情報

4. 基本的な使い方

新規テーマの作成

  1. 新規テーマの開始
    • 「ファイル」→「新規テーマ」を選択
    • またはツールバーの「新規」ボタンをクリック
    • キーボードショートカット: Ctrl+N
  2. テーマ名の設定

    テーマ名入力ダイアログが表示されます。わかりやすい名前を入力してください。

  3. ベーステンプレートの選択
    • 空のテーマ: ゼロから作成
    • ライトテーマ: 明るい色調のベース
    • ダークテーマ: 暗い色調のベース
    • ハイコントラスト: アクセシビリティ重視

テーマの保存

  • 基本的な保存: Ctrl+S
  • 名前を付けて保存: Ctrl+Shift+S
ヒント: 作業内容は定期的に自動保存されます。予期しない終了時でも、次回起動時に復旧オプションが表示されます。

5. テーマエディター

テーマエディターは、テーマの色、フォント、プロパティを直感的に編集するためのメイン機能です。

色の設定

カラーピッカーの使用

  • カラーホイール: 直感的な色選択
  • RGB値入力: 数値での正確な指定
  • 16進値入力: #RRGGBB形式での指定
  • HSV値入力: 色相・彩度・明度での指定

設定可能な色の種類

  • プライマリ色: メインの色調
  • セカンダリ色: アクセント色
  • 背景色: ウィンドウの背景
  • テキスト色: 文字の色
  • ボーダー色: 境界線の色
  • 選択色: 選択時のハイライト

フォントの設定

フォント選択オプション

  • フォントファミリー: システムフォントから選択
  • フォントサイズ: ポイント単位またはピクセル単位
  • フォントスタイル: 太字、斜体、下線、取り消し線

用途別フォント設定

  • メニューフォント: メニューバー、コンテキストメニュー
  • ボタンフォント: プッシュボタン、チェックボックス
  • ラベルフォント: 静的テキスト表示
  • 入力フォント: テキストフィールド、テキストエリア
  • リストフォント: リスト、ツリー、テーブル

プロパティの編集

プロパティエディターでは、テーマの詳細な設定を行えます。

プロパティの種類

  • 色プロパティ: 背景色、文字色、ボーダー色など
  • サイズプロパティ: 幅、高さ、マージン、パディング
  • フォントプロパティ: フォントファミリー、サイズ、スタイル
  • 効果プロパティ: 影、グラデーション、透明度

6. ゼブラパターンエディター

ゼブラパターンエディターは、WCAG準拠のアクセシブルなテーマを作成するための専用機能です。

WCAG準拠について

WCAG（Web Content Accessibility Guidelines）は、Webコンテンツのアクセシビリティに関する国際的なガイドラインです。

アクセシビリティレベル

  • レベルAA（推奨）
    • コントラスト比: 4.5:1以上（通常テキスト）
    • コントラスト比: 3:1以上（大きなテキスト）
  • レベルAAA（最高レベル）
    • コントラスト比: 7:1以上（通常テキスト）
    • コントラスト比: 4.5:1以上（大きなテキスト）

コントラスト比の計算

システムが自動的にコントラスト比を計算し、WCAG適合レベルを表示します。

判定基準の色分け

  • 緑色: WCAG AAA準拠（7:1以上）
  • 黄色: WCAG AA準拠（4.5:1以上）
  • 赤色: 基準未満（改善が必要）

色の改善提案

コントラスト比が不十分な場合、システムが自動的に改善案を提案します。

改善方法

  • 明度調整: 色の明るさを調整
  • 彩度調整: 色の鮮やかさを調整
  • 色相変更: 近い色相への変更
  • 代替色提案: 全く異なる色の提案

7. プレビューシステム

ライブプレビューシステムでは、作成中のテーマが実際のQtアプリケーションでどのように表示されるかを確認できます。

プレビュー機能

  • リアルタイム更新: テーマ変更時の自動反映（500ms以内）
  • 包括的なウィジェット表示: 主要なQtウィジェットを網羅
  • 状態テスト: 有効/無効、選択/非選択状態の確認

表示ウィジェット

プレビューでは以下のウィジェットが表示されます：

  • QPushButton（通常、無効状態）
  • QLineEdit（通常、フォーカス状態）
  • QComboBox（開閉状態）
  • QCheckBox、QRadioButton
  • QSlider、QProgressBar
  • QListWidget、QTreeWidget、QTableWidget
  • QTabWidget、QGroupBox

プレビュー画像のエクスポート

作成したテーマのプレビュー画像を保存できます。

対応形式

  • PNG: 高品質、透明度対応（推奨）
  • JPEG: 小サイズ、写真向け
  • BMP: 無圧縮、高品質
  • SVG: ベクター形式、拡大縮小対応

8. テーマ管理

テーマギャラリー

保存されたテーマを一覧表示し、効率的に管理できます。

アクセス方法

  • 「ツール」→「テーマギャラリー」
  • キーボードショートカット: Ctrl+G

表示形式

  • サムネイル表示: 大、中、小サイズでの表示
  • リスト表示: テーマ名、作成日、サイズを表示

検索・フィルタリング

  • 名前検索: テーマ名での検索
  • タグ検索: 設定されたタグでの検索
  • カテゴリフィルター: ライト、ダーク、ハイコントラスト
  • アクセシビリティレベル: AA、AAA準拠での絞り込み

最近使用したテーマ

最近編集したテーマに素早くアクセスできます。

  • 「ファイル」→「最近使用したテーマ」
  • 最大10個のテーマが記録されます
  • キーボードショートカット: Ctrl+1〜9

9. インポート・エクスポート

対応形式

形式 インポート エクスポート 用途
JSON ○ ○ Qt-Theme-Studio標準形式
QSS ○ ○ Qt Style Sheet形式
CSS ○ ○ Webアプリケーション用
Python × ○ プログラムコード形式
C++ × ○ Qt C++アプリケーション用

エクスポート手順

  1. 「ファイル」→「エクスポート」を選択
  2. 出力形式を選択
  3. 保存場所とファイル名を指定
  4. エクスポート設定を調整（オプション）
  5. 「保存」をクリック

インポート手順

  1. 「ファイル」→「インポート」を選択
  2. インポートする形式を選択
  3. ファイルを選択
  4. インポート設定を調整（オプション）
  5. 「開く」をクリック
注意: インポート時に現在のテーマは上書きされます。必要に応じて事前に保存してください。

10. アクセシビリティ機能

色覚サポート

  • 色覚異常シミュレーション: 異なる色覚での表示確認
  • 推奨色の組み合わせ: 色覚に配慮した色選択
  • パターンと形状の活用: 色のみに依存しない情報伝達

視認性の向上

  • 高コントラスト設定: 視認性を重視した色設定
  • 大きなフォント対応: 読みやすいフォントサイズ
  • 明確な境界線: 要素の区別を明確にする

操作性の向上

  • キーボードナビゲーション: マウスなしでの操作
  • フォーカス表示: 現在の操作位置を明確に表示
  • 適切なクリック領域: 操作しやすいボタンサイズ

12. トラブルシューティング

起動時の問題

アプリケーションが起動しない

  1. Qtフレームワークの確認 
    python -c "import PySide6; print('PySide6 OK')"
  2. qt-theme-managerの確認 
    python -c "import qt_theme_manager; print('qt-theme-manager OK')"
  3. Pythonバージョンの確認 
    python --version

    Python 3.8以上が必要です。

「Qtフレームワークが見つかりません」エラー

以下のコマンドでQtフレームワークをインストールしてください：

pip install PySide6

テーマ編集の問題

色が正しく表示されない

  • プレビューを手動更新: F5
  • ディスプレイの色設定を確認
  • 色空間設定（RGB、HSV）を確認

プレビューが更新されない

  • 設定で自動更新が有効か確認
  • メモリ不足の場合は他のアプリケーションを終了
  • アプリケーションを再起動

ファイル操作の問題

テーマファイルが開けない

  • ファイル形式が対応形式（JSON、QSS、CSS）か確認
  • ファイルの読み取り権限があるか確認
  • ファイルが破損していないか確認
  • UTF-8エンコーディングで保存されているか確認

保存ができない

  • 保存先フォルダの書き込み権限を確認
  • 十分なディスク容量があるか確認
  • ファイル名に使用できない文字が含まれていないか確認

パフォーマンスの問題

動作が重い

  • メモリ使用量をタスクマネージャーで確認
  • プレビューの更新間隔を調整（設定画面）
  • GPU加速を有効にする（設定画面）
  • Undo/Redo履歴の上限を下げる（設定画面）

設定のリセット

問題が解決しない場合は、設定をリセットしてください：

  1. アプリケーションを終了
  2. 設定ファイルを削除:
    • Windows: %USERPROFILE%\.qt_theme_studio\settings.json
    • macOS/Linux: ~/.qt_theme_studio/settings.json
  3. アプリケーションを再起動

サポート

問題が解決しない場合は、以下のリソースをご利用ください：

  • GitHub Issues: バグレポートや機能要望
  • ドキュメント: 最新のドキュメントを確認
  • ログファイル: ~/.qt_theme_studio/logs/ の内容を確認
